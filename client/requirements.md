## Packages
react-markdown | For rendering lesson content which is stored as Markdown
framer-motion | For smooth page transitions and micro-interactions
clsx | Utility for constructing className strings conditionally
tailwind-merge | Utility for merging Tailwind CSS classes safely

## Notes
The app uses a specific educational aesthetic (Preet English).
Authentication is assumed to be handled via the backend/auth flow, defaulting to a mock user context if explicit auth endpoints aren't fully integrated yet.
Images for lessons are dynamic; if not provided, fallback placeholders will be used.
